Introduction

1.1 Purpose

This (ConGratZ) is a website based on a greeting card/business card sharing model, where users can create and share their templates for other users.
This documentation describes version 1.0 of the website (The interface) while also referring to future functions not yet implemented.

Overall Description

2.1 Product perspective

ConGratZ is a stand-alone website

2.2 Product Functions

  • Create and publish Greeting/Business Cards and Templates
  • Browse User-made Card Templates
  • Search for User-made Card Templates

2.3 User Classes and Characteristics

There are three major user classes:
Firstly, business people who want to create cards for their business and can't hire a professional designer.
Second user class would be Middle aged women who want to send a greeting card to their relatives on social media, and need an easy to use solution.
And lastly, people who make funny greeting cards and want likes.

External Interface Requirements

3.1 User Interfaces

The card creator

The card creator consists of a central frame where the items for the card will be placed, a selector for the actual items, and buttons for the categories of items.

The buttons will toggle the selector window content. The buttons are as follows:
Frame button, which will toggle in the selector frames and borders to surround the card.
Image button, which will toggle in the selector a multitude of photos to choose from and place in the central frame.
The first item of the image selector is a button to upload your own photos Background button, which will toggle in the selector choices for the background.
Text button which will change the selector into text options
The selector is a window controlled by the buttons, where items will be displayed in a column. To see all the items, this window implements a scroll bar.
The central frame is where all the items go. These items can be selected and manipulated with the cursor. This will be shown by a border around the selected object. When text is selected, the user will be able to type and modify it.

The card viewer

The main role of the Card Viewer was to display a greeting card and some features that I will describe in the System Features. There are 3 standard buttons on the CardViewer page like“LogIn”, “SignUp”, “NewCard” and functions like “Search”, “Share” and “Download”. In the search section you can look up for a specific kind of greeting card in order to find what you need, also Download and Share become useful if you want to Download the greeting card and print it, or you can share it with your friends, online.

System Features

4.1 Card Creator

4.1.1 Description

The card creator is the Primary feature of the website. The priority is High. This feature enables users to create greeting cards or business cards using the two pannels on the right (the selector and the buttons). Users can create a greeting/business card.

4.2 Card Viewer

4.2.1 Description

Specific to card viewer page we have the following features:

  • Views: this will display how many users have seen the greeting card/bussines card;
  • Time: this will display the hours/days/monts or even years since the greeting card was posted which will be modified automatically;
  • Likes: it will show the rating of the greeting card;
  • Share button;
  • Download: if you like a greeting card/bussines card from the page you can download it;
  • Create Bussines Card and XML file: on the Card Viewer page you have two specific buttons to create those types of files in order to recommend yourself;
  • Report: user can give his feedback about the card (e.g.: if he find it offensive).
