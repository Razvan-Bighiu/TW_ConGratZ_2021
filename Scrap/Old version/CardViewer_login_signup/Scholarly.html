User interface

The main role of the Card Viewer was to display a greeting card and some features that I will describe in the System Features. There are 3 standard buttons on the CardViewer page like “LogIn”, “SignUp”, “NewCard” and functions like “Search”, “Share” and “Download”. In the search section you can look up for a specific kind of greeting card in order to find what you need, also Download and Share become useful if you want to Download the greeting card and print it, or you can share it with your friends, online.

System Features

Specific to card viewer page we have the following features:

  • Views: this will display how many users have seen the greeting card/bussines card;
  • Time: this will display the hours/days/monts or even years since the greeting card was posted which will be modified automatically;
  • Likes: it will show the rating of the greeting card;
  • Share button;
  • Download: if you like a greeting card/bussines card from the page you can download it;
  • Create Bussines Card and XML file: on the Card Viewer page you have two specific buttons to create those types of files in order to recommend yourself;
  • Report: user can give his feedback about the card (e.g.: if he find it offensive).
